## Key Features

- **Ingredient-Based Recipes:** Find meals using ingredients you already have at home  
- **YouTube Recipe Suggestions:** Get YouTube recipe links for your ingredients  
- **Food Expiry Tracker:** Track grocery expiry dates and receive reminders before items expire  
- **Grocery Management:** Add, edit, and delete ingredients easily  
- **Personalized Experience:** Save favorite recipes and manage your pantry efficiently  
- **Analytics & Insights:** Visualize your food usage and reduce waste over time  
- **Responsive & Intuitive Design:** Optimized for mobile screens for smooth experience
